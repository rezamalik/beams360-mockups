# BEAMS360 Mockups

Password-gated preview of the BEAMS360 UX redesign — four reference modules (Admissions, CRM, SIS, Fee Invoices) built around a shared task-first design system.

**Live preview:** https://rezamalik.github.io/beams360-mockups/

Password required. Contact the owner for access.

## What's inside

- `admissions.html` · `crm.html` · `sis.html` · `fees.html` — the four module mockups
- `index.html` — the password gate (landing page)
- `home.html` — mockup launcher after authentication

## Navigation

Inside any mockup:
- **⌘K / Ctrl+K** — open the command palette (search modules, actions, records)
- **9-dot waffle icon** in the top bar — full module launcher
- **Sidebar** — local navigation within the current module
